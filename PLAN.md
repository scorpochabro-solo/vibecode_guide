## 2. Контент-аутлайн (заголовки + тезисы)

### Блок 0 — Введение

**Hero.**
- H1: «За один вечер — от идеи до работающего продукта».
- Подзаголовок: «Пошаговый гайд по AI-агентам Claude Code и Codex для тех, кто никогда не писал код. Без опыта. Без страха. На Windows».
- CTA-1 «Путь Claude →» (коралл), CTA-2 «Путь Codex →» (минт).
- Строка-бейдж: `Проверено: {дата сборки}` (JetBrains Mono).

**«Как это вообще работает» (3 карточки bento).**
1. *AI-агент* — «сотрудник, который сам пишет код»: вы описываете задачу словами, агент создаёт файлы, пишет код, исправляет ошибки и показывает результат.
2. *Промпт* — «техзадание словами»: обычный текст, в котором вы объясняете, что хотите получить.
3. *Качество промпта = качество продукта* — чем конкретнее задача, тем лучше результат; этому и учит гайд.

**«Что понадобится» (4 пункта-строки).**
- Компьютер с Windows 10 или 11 (для Codex рекомендован Windows 11 — по офиц. докам).
- Аккаунт Claude и/или ChatGPT.
- План: для Claude Code — платный (Pro и выше; в Free его нет). Для Codex — формально входит во все планы, включая Free (ограниченно, «на попробовать»); для реальной работы — Plus и выше. Ссылки на официальные страницы цен, без цифр (по RESEARCH.md §2, §5).
- 1–2 часа времени и одна идея.

**Выбор пути.** Мини-таблица (3–4 строки: «где живёт», «какой план нужен», «кому подойдёт») → две кнопки-якоря.

### ЧАСТЬ 1 — CLAUDE

**Глава 1.1 «Регистрация и приложение».**
Шаги: аккаунт на claude.ai (вход по Google или ссылке из письма — пароля нет) → оформление плана (ссылка на цены) → установка приложения Claude для Windows (claude.com/download, x64) → обзор интерфейса: вкладки Chat / Cowork / Code, Projects. Callout: вкладка Code требует Git for Windows (RESEARCH.md §2); «ничего страшного, если интерфейс чуть отличается — он обновляется». 4 слота скриншотов.

**Глава 1.2 «Первые промпты» (практикум 15 минут).**
Живой пример: «сделай красиво» vs структурированное ТЗ (два блока рядом, плохой/хороший). Правило четырёх частей: контекст → задача → критерии → формат ответа. 3 мини-упражнения в чате.

**Глава 1.3 «Проект "Генератор промптов"».**
Создать Project → открыть Project instructions → вставить системный промпт (раздел 6.1 ТЗ, блок с кнопкой «Скопировать») → проверить на тестовой идее. Объяснение механики: сырая идея на входе — профессиональный промпт для Claude Code на выходе. Callout-бонус: Projects доступны даже на Free (до 5 проектов) — этот шаг можно сделать до оплаты. 2 слота скриншотов.

**Глава 1.4 «Установка Claude Code на Windows».**
Микро-урок «Что такое PowerShell и как его открыть» (Win+X → Терминал) → команда установки (из RESEARCH.md) → перезапуск терминала → `claude --version` → `mkdir C:\projects\my-first-site; cd C:\projects\my-first-site` → первый запуск `claude` → вход через браузер. Блок «Если что-то пошло не так»: `claude doctor`, «claude is not recognized» (перезапустить терминал/PATH), Alt+V для скриншотов. 5 слотов скриншотов.

**Глава 1.5 «Плагины: прокачиваем агента».**

**Глава 1.6 «Конвейер "Идея → Продукт"»** (центральная).

## 3. Дизайн-токены (финал)

Концепт: **premium developer editorial** — тёмная тема, дорогое техническое медиа. Tailwind v4 `@theme`:

```css
@theme {
  /* поверхности */
  --color-bg: #0B0C0E;
  --color-surface-1: #121316;
  --color-surface-2: #1A1C20;
  --color-border: rgba(255,255,255,0.08);
  /* текст */
  --color-text: #F4F4F2;
  --color-text-muted: #9BA0A8;
  /* акценты */
  --color-claude: #E8794A;      /* часть 1, тёплый коралл */
  --color-claude-dim: #B95F3D;
  --color-codex: #4AE8B0;       /* часть 2, холодный минт */
  --color-codex-dim: #3DB98F;
  --color-neutral-accent: #7C8CF8; /* общие блоки */
  /* геометрия */
  --radius-sm: 12px; --radius-md: 16px; --radius-lg: 24px;
  /* шрифты (next/font, cyrillic) */
  --font-display: 'Unbounded';     /* display-заголовки */
  --font-sans: 'Onest';            /* текст и UI */
  --font-mono: 'JetBrains Mono';   /* код, команды, бейджи, номера глав */
}
```

Активная часть задаётся CSS-переменной `--part-accent` на секции (коралл/минт/нейтральный) — все интерфейсные детали (подсветка TOC, бордеры при hover, номера глав) наследуют её; переход — плавный `transition` по цвету.

**Обязательные приёмы:** терминальные код-блоки (рамка окна, три точки, метка `PowerShell`, кнопка «Скопировать» → «✓ Скопировано»); крупные моно-номера глав (`01`…`07`); bento-сетки; noise-оверлей (opacity ≤ 0.04); 1–2 мягких glow в цвет активной части; прогресс-скроллбар; sticky-TOC.

**Motion (Framer Motion + Lenis):** секции — fade + translateY 24px, 0.6s, ease `[0.22,1,0.36,1]`, stagger 0.08s, viewport once; hover карточек — translateY(-4px) + акцентный бордер 0.25s; hero — построчное появление заголовка + лёгкий параллакс glow. `prefers-reduced-motion` — всё отключается, включая Lenis.

**Запреты:** стоковые фото, эмодзи вместо иконок (lucide-react), карусели, автоплей, >2 акцентов на экран, строки >70ch, «шаблонный SaaS».

Контраст: `#E8794A` на `#0B0C0E` ≈ 6.7:1, `#4AE8B0` ≈ 11.9:1, `#7C8CF8` ≈ 5.9:1, `#9BA0A8` ≈ 7.5:1 — все ≥ 4.5:1. Приглушённые варианты (`-dim`) — только для декоративных элементов, не для текста.
